## 📌 Project Overview

Wishlink is India's leading creator-commerce platform where influencers curate shoppable product wishlists and drive purchases across Amazon, Flipkart, Myntra, and more. This project builds a complete analytics pipeline to answer the core business question:

> **Which creators drive the most value — and how can Wishlink scale creator-led revenue?**

### What I Built
- **Python ETL pipeline** — cleans and transforms raw transaction data into analysis-ready tables
- **SQL queries** — aggregates KPIs, segments creators, computes cohort retention and lift scores
- **Category Lift metric** — measures how much better each creator converts vs the category average
- **Cohort Retention analysis** — tracks long-term user re-engagement across 8 monthly cohorts
- **Interactive HTML dashboard** — 5-panel drill-down with Chart.js, hosted on GitHub Pages
- **Tableau dashboard** — executive-level visualization of all key metrics
- **A/B test proposals** — three experiment designs grounded directly in the data findings

## 🧪 A/B Test Proposals

### Experiment 1 — Creator Category Page Pinning
Place high-lift creators on their best-fit category pages and measure incremental purchases.
- **Control:** Default category page
- **Treatment:** Top-lift creator featured on category landing page
- **Primary metric:** Purchases per 1,000 category impressions
- **Target pairs:** Creator #15 × Toys (3.31×), Creator #49 × Home (3.14×), Creator #9 × Food (2.68×)

### Experiment 2 — Creator Badge on Product Page
Add a "Recommended by [Creator]" badge on products promoted by high-lift creators.
- **Primary metric:** Add-to-cart rate
- **Guardrail:** Return rate, average order value

### Experiment 3 — Lift-Ranked Creator Feed
Show users a creator feed ranked by lift score instead of chronological order.
- **Primary metric:** Purchases per weekly active user
- **Guardrail:** Session length, creator page CTR

## ▶️ How to Run Locally
```bash
# 1. Clone the repo
git clone https://github.com/rohitmannur007/wishlink-creator-commerce-analytics.git
cd wishlink-creator-commerce-analytics

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run ETL pipeline
python scripts/etl.py

# 4. Generate Tableau-ready CSVs
python scripts/aggregates_for_tableau.py

# 5. Build cohort data
python scripts/make_cohort.py

# 6. Load into SQLite (optional)
python scripts/load_to_sqlite.py

# 7. Open the dashboard
open index.html
```
